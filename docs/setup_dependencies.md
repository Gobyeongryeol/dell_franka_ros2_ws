# Setup Dependencies

## Required Environment

- Ubuntu 22.04
- ROS2 Humble
- MoveIt2
- Franka ROS2 stack
- Intel RealSense ROS2 package
- CUDA 12.4
- Miniforge/Conda
- AnyGrasp SDK
- AnyGrasp checkpoints/license
- YOLO weights

## Not Included in This Repository

The following files and directories are intentionally not included in this GitHub repository:

- AnyGrasp SDK
- AnyGrasp checkpoints
- AnyGrasp license files
- YOLO weight/model files
- datasets
- debug images
- ROS bag files
- ROS2 `build/`, `install/`, `log/`

Install or copy these dependencies locally before running the full pipeline.
